VIBE CODING — Presentasi Interaktif (HTML, gaya Prezi / sci-fi)
================================================================

CARA MENJALANKAN
- Buka index.html langsung di browser (Chrome/Edge) pada Papan Pintar (IFP).
- Atau unggah seluruh folder ini ke hosting statis:
    * mahasgames.com/vibe/  (WAMP: taruh folder di dalam www)
    * GitHub Pages: push folder ini ke repo, aktifkan Pages.
- Seluruhnya OFFLINE — tidak butuh internet (kecuali saat membuka tautan proyek).

STRUKTUR
  index.html        -> aplikasi presentasi (semua CSS+JS di dalamnya)
  slides/           -> 12 gambar halaman (slide-01.png .. slide-12.png)

KONTROL
  Geser kiri/kanan  -> halaman berikut / sebelumnya  (mulai dari halaman 1)
  Cubit (2 jari)    -> zoom in / zoom out di titik mana pun
  Seret (1 jari)    -> menjelajah kanvas bebas (mode jelajah)
  Ketuk halaman     -> membuka TAUTAN aktif halaman di tab baru
  Tombol ▦ (Peta)   -> lihat semua halaman; ketuk untuk terbang ke sana
  Keyboard          -> ← → / Spasi, +/- zoom, O peta, F layar penuh, Home awal
